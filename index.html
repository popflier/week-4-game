The Crystal Game

Game Instructions:

At the beginning of each game you will see a random target number. Each of the crystals shown has a random numeric value assigned to it between 1 - 12. The object of the game is to continue clicking crystals to have the sum of your numbers match the target number. If you are able to match the exact total then you win. Go over and you lose.

Simple enough. Let's begin!
